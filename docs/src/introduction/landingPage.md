| 6.    	| Investments link          	| A link to the Investments Module.                                                                                                                 	|
| 7.    	| Workflow link             	| A link to the Workflow Module.                                                                                                                    	|
| 8.    	| Fees and Commission link  	| A link to the Fees and Commission Module.                                                                                                         	|
